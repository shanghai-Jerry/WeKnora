# WeKnora 版本更新说明

## 0.3.4 → 0.3.5 主要更新

### 🚀 新功能
- **IM 集成扩展**：新增 Telegram、DingTalk、Mattermost 机器人适配器，支持 Webhook 和长轮询模式
- **IM Slash 命令系统**：内置 /help、/info、/search、/stop、/clear 五个命令
- **IM 分布式协调**：基于 Redis 的多实例协调，支持队列限流、全局限流、消息去重
- **建议问题**：基于知识库生成建议问题，在对话中展示
- **VLM 自动描述 MCP 工具图片**：MCP 工具返回图片时自动生成文字描述
- **新模型提供商**：新增 Novita AI
- **渠道追踪**：新增渠道字段追踪知识条目的来源（web/api/im/浏览器扩展）

### ⚡ 优化
- MCP 工具名称从 UUID 改为 service.Name，支持重连后稳定调用
- 增强意图分类和上下文模板
- 知识搜索增强：SQL LIKE 通配符转义、标题过滤、URL 和 HTML 文件支持
- MiniMax 模型从 M2.1 升级到 M2.7
- DingTalk AI Card 流式输出优化

### 🐛 修复
- 修复 Agent 无知识库时的空响应问题
- 修复 UTF-8 字节截断导致的中文/emoji 错误
- 修复 marked.js 使用错误
- 修复 vLLM 流式推理内容
- 修复全局默认存储引擎未生效问题
- 修复 API key 加密更新丢失问题

### 🔒 安全增强
- 增强 SSRF 保护，使用 SSRFSafeDialContext

---

## 0.3.5 → 0.3.6 主要更新

### 🚀 新功能
- **ASR 语音识别**：集成 ASR 模型，支持音频文件上传和转录
- **飞书数据源自动同步**：完整的数据源管理，Wiki/Drive 自动同步
- **OIDC 认证**：支持 OpenID Connect 登录
- **IM 引用/回复上下文**：提取引用消息并注入 LLM 提示
- **基于线程的 IM 会话**：支持多用户在消息线程中协作
- **文档摘要**：AI 生成文档摘要
- **Tavily 搜索**：新增网络搜索提供商
- **MCP 自动重连**：连接丢失时自动重连
- **并行工具调用**：Agent 模式并发执行多工具
- **@提及范围限制**：限制用户 @提及到允许的知识库范围

### ⚡ 优化
- 优化登录页面渲染性能
- 统一 NVIDIA API 支持聊天和 VLM 模型
- Prompt 语言回退使用 WEKNORA_LANGUAGE 环境变量
- Markdown 表格分块时保留表头
- Elasticsearch ID 字段动态处理
- 知识标签批量更新增加授权检查
- 数据库查询工具增强

### 🐛 修复
- 修复飞书群聊 @提及未剥离导致命令失败
- 修复 SSEReader 返回错误类型
- 修复删除知识时图片未清理
- 修复远程图片解析
- 修复 mermaid 刷新不渲染
- 修复知识文档分类无法取消选择

## 总结

从 0.3.4 到 0.3.6，项目主要在以下方面持续迭代：

1. **IM 生态完善**：从 0.3.5 开始大规模扩展 IM 集成，从单一平台扩展到 Telegram、DingTalk、Slack、飞书等多个平台，并实现了 Slash 命令、分布式协调等企业级功能
2. **AI 能力增强**：新增 ASR 语音识别、文档摘要、VLM 图片描述等能力
3. **企业级功能**：新增 OIDC 认证、数据源同步、IM 线程协作等
4. **安全性提升**：持续增强 SSRF 防护、API key 加密等安全特性
